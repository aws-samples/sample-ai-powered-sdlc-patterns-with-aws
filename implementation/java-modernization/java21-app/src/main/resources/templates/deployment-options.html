Ready-to-Use Deployment Scripts

All deployment scripts are available in the project's deployment-scripts/ directory:

🚀 Quick Start Commands
# Make scripts executable
chmod +x deployment-scripts/*.sh

# Deploy to App Runner (Recommended)
./deployment-scripts/app-runner-deploy.sh

# Deploy to ECS Fargate
./deployment-scripts/ecs-fargate-deploy.sh

# Deploy to Lambda
./deployment-scripts/lambda-deploy.sh
📋 Available Scripts

  • ecs-fargate-deploy.sh

  • ec2-deploy.sh

Prerequisites
  • AWS CLI configured with appropriate permissions
  • Docker installed (for container-based deployments)
  • EB CLI installed (for Elastic Beanstalk deployment)
